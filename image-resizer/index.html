FAQs

Can I keep the original aspect ratio?

Yes, keep-aspect mode automatically updates the other dimension as you type.

Which formats are supported?

PNG, JPG, and WEBP exports are supported in modern browsers.

Does this reduce file size too?

Usually yes, especially when resizing to smaller dimensions or exporting JPG/WEBP.
